# 👋 Discord Welcome Bot

A simple and customizable **Discord Welcome Bot** built with **Node.js** and **discord.js**.  
This bot automatically welcomes new members when they join your Discord server.

---

## 🚀 Features

- 👋 Welcome message when a member joins
- 📢 Send welcome message to a specific channel
- 🎭 Auto role for new members
- ✉️ Optional DM welcome message
- ⚡ Lightweight and easy to deploy
- 🔧 Fully customizable configuration
